Qassida

Générateur de poèmes arabes avec Pytorch

        
        Starkus ~ $

Installation

  • Windows
            $  pip install -r requirements.txt
            $  python qassida.py -firstWord [premier mot du poème]

C'est quoi ?

Le code permet de générer des poèmes dans un style donné en utilsant des Réseaux de neurones récurrents (RNN)

Livrables

Le code peut etre importé et exécuté localement, où testé sur le site web

Outils

IDE + Python 3.x + Pytorch
